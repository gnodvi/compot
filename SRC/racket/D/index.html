PS/Tk

A Portable Scheme Interface to Tk

PS/Tk is a portable interface between the Scheme programming language and the Tk Toolkit. Except for a very small portion of code it is written in pure R5RS Scheme.

All you need to write GUI applications in Scheme is Tcl/Tk (and optionally Tile), a supported Scheme implementation, and PS/Tk.

PS/Tk runs on these implementations: Bigloo, (Petite) Chez, Chicken, Gambit, Gauche, Guile, Ikarus, Kawa, Larceny, MzScheme, Scheme 48, Scsh, Stklos.
